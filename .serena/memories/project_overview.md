# プロジェクト概要

## プロジェクト名
**井原誠斗 ポートフォリオサイト**

## 目的
- フリーランスWeb開発者としてのスキルと実績をアピール
- AI技術とWeb開発の融合による差別化
- 潜在的なクライアントとの接点を創出
- 技術的専門性の実証

## ターゲットユーザー
1. **中小企業の経営者・決裁者**: 自社のWebサイト・アプリ開発を検討している
2. **スタートアップ企業**: MVP（最小実行可能製品）の開発が必要
3. **既存企業のWeb担当者**: システムの現代化・リニューアルを検討

## プロジェクトの特徴
- **最新技術スタック**: Next.js 15 + React 19 + TypeScript
- **モダンUI**: Tailwind CSS + shadcn/ui
- **プロフェッショナル機能**: Resendによるメール機能
- **レスポンシブデザイン**: 全デバイス対応
- **SEO最適化**: 検索エンジンからの流入確保

## プロジェクト構造
- **App Router**: Next.js 15のApp Routerベース
- **コンポーネント設計**: shadcn/uiの再利用可能なUIコンポーネント
- **Server Actions**: フォーム送信などのサーバーサイド処理
- **型安全性**: TypeScriptによる厳格な型管理

## 主要機能
1. **ヘッダー・ナビゲーション**: 固定ヘッダー、スムーズスクロール
2. **ヒーローセクション**: インパクトのあるファーストビュー
3. **Aboutセクション**: 開発者としての経験と専門性
4. **Servicesセクション**: 提供可能なサービスの明確な説明
5. **Skillsセクション**: 技術スタックの具体的な提示
6. **Portfolioセクション**: 実績の視覚的な紹介（準備中）
7. **Contactセクション**: お問い合わせフォームとメール送信機能

## 成功指標（KPI）
- **月間ユニークビジター数**: 500人以上
- **お問い合わせ件数**: 月5件以上
- **サイト滞在時間**: 平均3分以上
- **バウンス率**: 60%以下
