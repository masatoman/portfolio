# 技術スタック

## フロントエンド
- **Next.js 15**: App Routerベースのフルスタックフレームワーク
- **React 19**: ユーザーインターフェース構築
- **TypeScript 5**: 型安全なJavaScript

## スタイリング
- **Tailwind CSS 3.4**: ユーティリティファーストCSSフレームワーク
- **shadcn/ui**: 再利用可能UIコンポーネントライブラリ
- **Radix UI**: アクセシブルなプリミティブコンポーネント
- **tailwindcss-animate**: アニメーションプラグイン

## フォント
- **Geist Sans**: モダンなサンセリフフォント
- **Geist Mono**: 等幅フォント

## アイコン・UI要素
- **Lucide React**: アイコンライブラリ
- **sonner**: トースト通知
- **class-variance-authority**: バリアント管理
- **clsx + tailwind-merge**: クラス名の結合とマージ

## フォーム管理
- **React Hook Form**: フォーム状態管理
- **Zod**: スキーマバリデーション
- **@hookform/resolvers**: フォームバリデーション統合

## バックエンド・サービス
- **Next.js Server Actions**: サーバーサイド処理
- **Resend**: メール送信サービス

## その他のライブラリ
- **date-fns**: 日付操作
- **recharts**: チャートライブラリ
- **embla-carousel-react**: カルーセルコンポーネント
- **input-otp**: OTP入力コンポーネント
- **react-resizable-panels**: リサイズ可能なパネル
- **vaul**: ドロワーコンポーネント

## 開発・ビルドツール
- **pnpm**: パッケージマネージャー（高速で効率的）
- **PostCSS**: CSS処理
- **Autoprefixer**: ベンダープレフィックスの自動付与
- **ts-node**: TypeScript実行環境

## デプロイ・ホスティング
- **Vercel**: ホスティングプラットフォーム（予定）
- **ドメイン**: ihara-frontend.com

## 開発環境要件
- **Node.js**: v18以上
- **パッケージマネージャー**: pnpm
- **OS**: Darwin (macOS)、ただしクロスプラットフォーム対応
