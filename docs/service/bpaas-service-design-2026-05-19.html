2. サービス別 詳細設計

① 現場写真 整理代行

BPaaS ¥15,000/月 → SaaS ¥6,000/月 (40%)

代行内容: LINE で受信した現場写真を工事番号別に仕分け → 月末に PDF 台帳生成 (完了報告 / 施主アルバム / 保証書添付 の 3 テンプレ)

工程時間備考
写真受信 + 初期分類 (月 100-300 枚)1.5-2.5hEXIF + 顧客コメントで紐付け
工事番号確定 + タイトル付け30 分
PDF 生成 (テンプレ化済)30 分
顧客 LINE 返信 / 月末配送30 分
合計3-4h時給 3,750-5,000 円
⚠️ 推測根拠: 写真 1 枚 30 秒の処理時間は 推測。 実際は OCR (現場黒板) や工事番号推定の精度次第。 月 100-300 枚も 推測 (友人 N=1 で確認していない)。

② 領収書・経理代行

BPaaS ¥30,000/月 → SaaS ¥10,000/月 (33%)

代行内容: 領収書・請求書を LINE 写真送信 → OCR 仕訳 / 請求書発行 (月 10-20 件) / 入金消込 / 月次レポート (売上・経費・粗利)

工程時間備考
領収書 OCR + 仕訳 (50-100 枚)2-4hfreee / MoneyForward 投入
請求書発行 (10-20 件)2-3h顧客承認後送付
入金消込1-2h銀行明細チェック
月次レポート1h
合計6-10h時給 3,000-5,000 円
⚠️ 税理士法 52 条の壁 確認済: 仕訳・記帳代行は「経理事務代行」 として OK だが、 税務相談・申告書作成に踏み込むと無資格営業で違法。 線引きをパンフレット明記必須。 5/22 北原氏 Q12 で 「事務代行の範囲」 を確認推奨。
⚠️ 推測根拠: 月次の領収書 50-100 枚 / 請求書 10-20 件 は 推測。 5-30 人規模工務店の実際の業務量を 要裏取り (友人 N=2-3 + 業界統計 or freee 工務店事例)。

③ 日報作成代行

BPaaS ¥20,000/月 → SaaS ¥8,000/月 (40%)

代行内容: 現場で音声録音 (1 件 30 秒-2 分) → LINE 送信 → Whisper 文字起こし → Claude で整形 (現場名 / 作業内容 / 人工 / 翌日予定) → 日次 PDF + 月末まとめ

工程時間備考
音声受信 (日 1-3 件 × 月 25 日)(自動)50-75 件
AI 文字起こし後の整形 (1 件 5-10 分)4-12h主にここ
月末 PDF 製本1h
合計5-13h時給 1,500-4,000 円 (最低)
⚠️ 需要不確実 確認済: 友人 N=1 で「日報は紙」 と判明 (memory friend-hearing-result-2026-05-14)。 訴求軸は「紙より楽」 ではなく「日報書く 1 時間がもう一現場の見積に変わる」 へ。 5 サービスで撤退判定の可能性最有力。
⚠️ 推測根拠: 整形 1 件 5-10 分は 推測。 Whisper 精度と現場用語 (人工・歩掛・型枠等) のカスタム辞書次第で 2-15 分にブレる。

④ 見積書作成代行

BPaaS ¥30,000/月 → SaaS ¥12,000/月 (40%)

代行内容: 初回オンボーディングで過去見積 10-30 件をベクトル DB 学習 (5-10h、 1 回のみ)。 月次は新規案件依頼 LINE 受信 → 類似案件参照 → ドラフト生成 → 顧客確認 → 修正 → 確定。

工程時間備考
見積依頼受領 (月 5-15 件)1h
ドラフト生成 + 確認 (1 件 30 分)2.5-7.5h類似案件 3 件参照
顧客との往復修正1-2h
合計 (運用後)3.5-10h時給 3,000-8,500 円 (最高)
💡 価値訴求: 「半日 → 10 分」 の時短価値は工務店オーナーが理解しやすい。 BPaaS 単価が 5 サービス中で最高水準を維持できるのもこのため。 SaaS 化後も値崩れしにくい。
⚠️ 推測根拠: 月 5-15 件は 推測。 5-30 人規模工務店の実際の月次見積数を 要裏取り。 初回学習 5-10h も Claude + ベクトル DB の精度次第。

⑤ 顧客連絡代行

BPaaS ¥20,000/月 → SaaS ¥6,000/月 (30%)

代行内容: 工事進捗写真 + 一言コメント受信 → 施主向け定期メッセージ作成 (週 1-2 回) → LINE/メール/SMS 送信 → クレーム兆候検出 → 工務店エスカレ。

工程時間備考
進捗写真受信 (週 2-3 回)30 分
メッセージ作成 (月 8-12 件 × 10-15 分)2-3h
施主応対 (返信時)2-5h変動大
クレーム エスカレ判断30 分
合計4-8h時給 2,500-5,000 円
💡 主顧客の最痛点に直撃 確認済: 友人 N=1 で「個人客 9 割」 判明 (memory friend-hearing-result-2026-05-14)。 5 サービス中で B2C 接点を肩代わりするのはこれだけ → 最重要候補。
⚠️ クレーム責任分界: 施主との誤発信 → 工務店信用毀損のリスク。 契約書で「代行範囲」「最終判断は工務店」「賠償上限」 を明示必須。 5/22 北原氏 Q12 で「事業者損害保険」 込みで相談推奨。

3. 価格設計の根拠

BPaaS:SaaS 価格比 = 2.7-3.4 倍 推測

フルマネージド代行 vs セルフサービス SaaS の一般相場は 3-4 倍と言われているが、 工務店業界で同等の代行サービス相場を 要裏取り。 Bizer (経理代行) / Tanomu (受発注) / 経理プラス 等の月額調査未実施。

時給 3,000-5,000 円ライン 推測

masatoman の本業時給感覚 (受託 5,000-8,000 円/h) より低めだが、 副業 + 1 人運用なら許容。 ③ 日報代行のみ 1,500-4,000 円で要警戒。

BPaaS の「高価格」 が正当化される理由: 主顧客 (IT 弱者の工務店オーナー) は「自分で操作したくない」 という核心特性 (memory customer-target)。 SaaS は学習コスト障壁で受け入れ困難 → BPaaS で「LINE に送るだけ」 という入口を作る → 一度慣れると SaaS に動かない (粘着性) → BPaaS が安定収益源として残る、 という戦略的価格構造 ([[komuten-bpaas-ai-agent-strategy-2026-05-18]])。

5. 設計上の検証必要事項 (5/22 北原氏 / 友人 N=2-3 で確認)

  1. 税理士法 52 条のグレーゾーン線引き: 経理代行を「事務代行」 内に留めるための具体的線引きを 5/22 Q12 で相談
  2. 日報代行の需要: 友人 N=1 で「紙」 判明 → N=2-3 で再確認、 撤退判定の可能性
  3. クレーム責任分界: 顧客連絡代行で施主からの訴訟リスク発生時の責任を契約書でどう書くか
  4. 5 サービスの順位: 友人 Phase 3 ヒアリング 30 分 で「これ全部出されたらどれから欲しいか」 を順位付け
  5. 属人化: masatoman 1 人運用は病気・休暇で全停止、 5 顧客超えたら 副業外注 (シニア経理経験者 / 元現場監督) 検討
  6. 業界相場ベンチマーク: Bizer / Tanomu / 経理プラス / 写真整理代行 / 日報代行サービス の月額調査
  7. 工務店の実業務量: 領収書枚数 / 月次見積件数 / 写真枚数 を友人 N=2-3 で実数確認

6. ⚠️ 信ぴょう性の所在 — 事実 vs 推測 vs 要裏取り

✅ 確認済 (一次情報源あり)

  • masatoman 月稼働 60-80h: memory user-work-time-constraint 一次情報
  • 友人 N=1 ヒアリング結果: 買い切り嬉しい / 個人客 9 割 / 日報紙 / 横つながり弱 / メール 3 割不可 (memory friend-hearing-result-2026-05-14)
  • 税理士法 52 条: 税理士業務 (税務代理 / 税務書類作成 / 税務相談) の独占。 法令確認可能
  • 主顧客特性 (IT 弱者・自分で操作したくない): memory customer-target + issue-finder 235 件痛みクラスタ
  • BPaaS 戦略仮説: memory komuten-bpaas-ai-agent-strategy-2026-05-18 (5/18 別 claude.ai セッション)

⚠️ 推測 (一次情報源なし、 検証必要)

  • 5 サービスの労力試算 全て: 写真 1 枚 30 秒 / OCR 仕訳 1 件 2 分 / 整形 1 件 5-10 分 等は全部 masatoman + Claude の机上推測。 実測ゼロ
  • BPaaS 月額価格 ¥15K-30K: 業界比較なし、 freee 経理代行 / Bizer / Tanomu 等の月額調査ゼロ
  • SaaS 価格 = BPaaS の 30-40%: 「市場相場」 と書いたが出典なし
  • 月次業務量: 領収書 50-100 枚 / 請求書 10-20 件 / 見積 5-15 件 / 写真 100-300 枚 / 音声 50-75 件 は全部推測
  • クレーム発生率: 顧客連絡代行のリスク量定量化なし
  • 5 サービスの需要順位: 友人 N=1 でも個別質問していない

🔍 要裏取り (確認可能、 5/22 までに or 友人 N=2-3 で)

  • 業界代行サービス相場: WebSearch で Bizer / Tanomu / 経理プラス / 写真整理代行 / 日報代行 の月額調査 (1-2h で可能)
  • 税理士法 52 条の具体線引き: 国税庁サイト + 税理士会 Q&A、 5/22 Q12 で北原氏に確認
  • 1 人運用代行業の天井ライン: 業界統計 (中小企業庁 / 経済センサス)、 副業ガイドライン
  • 工務店の実際の月次業務量: 友人 N=2-3 ヒアリングで実数取得 (Phase 3 質問追加)
  • 事業者損害保険の費用感: 顧客連絡代行のクレームリスク対策、 5/22 Q12 ついでに北原氏に
