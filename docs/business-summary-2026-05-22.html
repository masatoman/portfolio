// 03 / 法人化方針 + 公庫融資 (2026-05-13/15 更新、 北原氏面談 主要相談テーマ)

方針確定 当面は個人事業主のまま継続 + 公庫融資は法人化より先
検討の結果、 2026/6 マイクロ法人設立予定を撤回。 工務店向け事業が軌道に乗ったタイミング (所得 900 万超目安) で改めて法人化を検討。
判断理由: 法人固定費 (年 7 万円〜) より検証フェーズの現金優先 / 小規模企業共済 (月 7 万 = 年 84 万控除) 実施済で追加節税を急ぐ必要なし / 軌道後の法人化が節税効果大

公庫融資 100 万円 (新規開業資金): 法人化すると審査が「法人としての実績ゼロ」 で再評価されるため、 個人事業主のうちに通す。 5/22 北原氏面談後 (5 月末〜6 月初旬) に申込予定。 既存受託契約書類 (顧客契約書 / 請書 / 通帳入金記録) を別紙添付で「実態あり」 評価を取りに行く。
5/22 で確認 「軌道」 の判断指標 + 信用度ルート (合同会社、 節税抜き) の要否
  • 軌道判断の指標 (月商 / 顧客数 / 所得など) は何にすべきか
  • 工務店オーナーから見て個人事業主だと信用度問題が出るか / 補強策
  • 不利と判明した場合、 節税効果なしの合同会社設立 (6-7 月) は妥当か
  • 将来的に IT 導入支援事業者として動きたい意向、 現時点で念頭に置くべき注意点 (詳細は商品確定後 10-12 月以降に再相談)

⚠️ 友人 (現場監督) N=1 ヒアリング (5/14 実施) で判明した前提崩壊 5 件:

  • 「買い切りの方が嬉しい」 → 月額モデル単独だと刺さらない可能性

これらの前提崩壊が、 BPaaS / AI エージェント ハイブリッド + アトツギ層拡張 + 主顧客像 再定義 (5/22 で議論) の検討動機。 N=4-5 まで増やしてから v3 で商品設計を確定する想定。
