// 06 / 法人化論点の整理 (2 論点に分離)

「マイクロ法人 vs 合同会社」 は実は対立軸ではなく、 独立した 2 つの論点:

✅ 確定済み
論点 1: マイクロ法人の設立 (2026/6 予定)
既存フリーランス本業 (英語等) の節税目的。 合同会社で設立、 役員報酬は社会保険最低限。 memory business-form-roadmap の方針通り。 これは確定。

重要: マイクロ法人は工務店ビジネスとは無関係。 既存本業の節税専用。
⏳ 検討中
論点 2: 工務店ビジネスの法人化
元々は「工務店相手に商売する想定なし」 だった (= マイクロ法人を考え始めた時)。 最近「工務店相手に仕事できるかも」 と思い始めた → 新しい論点として浮上。

選択肢:
  • A. 個人事業主のまま 工務店ビジネスを続ける (現傾向)
  • B. 別の合同会社 を工務店ビジネス専用に設立
  • C. マイクロ法人を工務店ビジネス用に転用 (節税専用スキームは崩れる)
現傾向: A (マイクロ法人 = 節税専用 / 工務店ビジネス = 個人事業主) の分離運用。 ただし工務店オーナーから見て個人事業主だと信用度に問題が出ないか不安 → 北原氏に相談。

// 07 / 質問 Q4 を 3 分割 (相談予約 #3 #4 に対応)

Q4-a (運用)
既存本業の節税目的でマイクロ法人 (合同会社) を 2026/6 に設立予定です。 工務店向け SaaS / 受託は別収入源として個人事業主のまま続ける想定ですが、 この分離運用に実務上の問題点 (経費按分・社会保険・税務調査リスク等) はありますか?
→ 相談予約 #3 法人化のタイミングに対応。 現傾向 A 案の妥当性確認。
Q4-b (信用)
工務店オーナーから見て、 取引相手が個人事業主か法人かで信用度や決裁プロセスに差が出ますか? 工務店ビジネスを伸ばすなら、 個人事業のままより別の合同会社化を急ぐべきでしょうか? (= 論点 2 の選択肢 B 案)
→ 相談予約 #1 (決裁プロセス) + #3 (法人化タイミング) に対応。
Q4-c (補助金)
「マイクロ法人 (節税専用) + 個人事業主 (工務店ビジネス)」 という分離運用の場合、 IT 導入支援事業者登録はどちらの主体で可能ですか? 個人事業主のままで申請できる仕組みはありますか?
→ 相談予約 #4 IT 導入支援事業者登録に直接対応。 memory business-form-roadmap の「マイクロ法人とは矛盾」 という現状記述を事実確認。
