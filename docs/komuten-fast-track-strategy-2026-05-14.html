01クリティカルパス 逆算

工務店契約サイクル の 典型:

  • 初回接触 (1 週) → 1 次商談 (1 週) → demo 持ち込み (1-2 週) → 社内検討 (2-3 週) → 契約 (1 週)
  • 合計: 6-8 週間

7 月末契約 から 逆算 すると:

初回接触 完了5/315 件分水嶺

demo 持ち込み6/303-5 件絞り込み
1 社目 契約6/301 件目標

クリティカル: 5/31 までに「初回接触 5 件並行」 状態 を 作れ ない と、 1 社の遅延 で 全体 が 止まる。 接触 → 商談 コンバージョン 30-40% を 前提 に 接触 機会 15-25 件 が 必要。

03盲点 5 つ — Ultrathink で 見えてきた もう 1 階層

前回 の 答え (紹介ハブ 多重化) は 正しい方向 だが、 さらに 深掘り すると 5 つ の 盲点 が 浮かぶ。 順に 潰す。

盲点 01

工務店 の 真の意思決定者 は 「アトツギ」 = 2 代目 3 代目

家族経営 が 大半 で、 意思決定 は 3 層 構造:

層人物ITmasatoman さん の 位置
決裁社長 (60 代)弱説得対象
経理社長 の 妻 (60 代)中金銭面 で 味方化
推進2 代目・3 代目 (30-40 代)強真の キーパーソン

アトツギ は 「父親 を 説得 する 材料」 として SaaS を 欲しがる。 IT 弱者 なのは 社長 だけで、 アトツギ は 普通に LINE 以外 も 使える。

使う べき 公的プログラム

  • アトツギ甲子園 (中小企業庁、 毎年 3 月決勝、 地域予選あり) — 多摩地区 アトツギ が 集まる、 工務店 割合 高い
  • 東京都事業承継・引継ぎ支援センター (東京商工会議所内) — アトツギ向け 相談窓口、 紹介ハブ として 強い
  • 後継者人材バンク (東京商工会議所) — アトツギ × 専門家 マッチング枠 で 潜入可能
  • アトツギベンチャー (中小企業基盤整備機構) — DX 系 アトツギ の LinkedIn 的 コミュニティ

5/22 北原氏面談 で 「父親世代 の 社長 より、 2 代目 を 窓口 に したい 工務店 ありませんか」 と 質問 の 角度 を 変える と 刺さりが 変わる。

盲点 02

信用補完 は 「二重化」 が 必須

個人事業主 の 最大の敵 = 「この人 で 大丈夫?」。 信用補完 は 1 つ だと 足りない。 2 経路 の 権威 を 借りる。

レイヤー A: 公的認定 (中期、 6-7 月)

  • 認定経営革新等支援機関 — 自身は取りにくいが、 多摩地区 の 認定支援機関 税理士 と 業務提携 → 「補助金申請 まで 一気通貫」 を 共同提案
  • DX 認定事業者 (経産省) — 個人事業主 取得可、 書類重い が 取れれば 名刺武器
  • IT 導入支援事業者 — 6 月公募 で 登録

レイヤー B: 地域ブランド (短期、 5-6 月)

  • 「たましん指定 SaaS」 / 「府中商工会議所 IT 化推進パートナー」 という 肩書き を 取りに行く
  • 正式 認定制度 では なく、 担当者ベース の 口約束 で 先行 する のが コツ
  • 「府中商工会議所 さんから 紹介 いただいた」 と 次の 工務店 に 言える 状態 を 作る

A + B の 二重化 で 「信用補完 の 壁」 が 崩れる。 A 単独 = 補助金代行業者 と 同じ で 競合多すぎ、 B 単独 = フリーランス で 信用不足。

盲点 04

補助金 の 解像度 — 実質負担 1/8 訴求 が 武器 (5/22 Q10)

工務店オーナー が 申請者 に なり、 自身 が 支援者 に なる 補助金:

盲点: 東京都 DX 推進助成金 が 前回回答 で 欠落 していた。 都内工務店専用、 1/2 上限 300 万、 SaaS 直接対象。 5/22 北原氏 Q10 に 「都の助成金 スケジュール」 確認 を 追加 すべき。

主顧客向け の 決まり文句

「IT 導入補助金 で 3/4 + 東京都 DX 助成金 で 1/2 残り = 実質負担 1/8 で 導入 できます」

これが 言える と 商談 コンバージョン 30 → 60% に 跳ねる 可能性。

盲点 05

業界紙 の 権威借用 (公的に近い)

工務店業界 には 強い 業界紙 がある。 取材露出 → 信用補完 → 紹介自動化 の 三段ロケット。

媒体読者層取材依頼 の 角度
新建ハウジング工務店経営者 (週刊)「IT 弱者工務店向け SaaS で 8 デモ 需要計測 した 個人事業主」
リフォーム産業新聞リフォーム工務店同上
建築知識 (エクスナレッジ)設計事務所 中心設計事務所経由ルート に 直結
日経ホームビルダー大手寄り信用補完 の 最強パッチ

取材 は 無料、 自薦可能。 demo-gallery + issue-finder 235 件分析 = 記事ネタ として 強い。 5 月中 に 1-2 媒体 に PR の 連絡 を 入れる 価値あり。

即効性 は 中程度 (取材 → 掲載 2-3 ヶ月)。 7 月末契約 には 間に合わない が、 8-9 月 の 2 社目以降 の 加速材料 として 効く。

05失敗パターン (回避 リスト)

Ultrathink で 見えてくる 「爆速 の 罠」 を 列挙。 これらに 引っかかると 7 月末 が 来ない。

✕公的施設 の セミナー 受講 で 時間 を 溶かす — 学習 < 実践 < 売る の 原則 (STRATEGY.md §4)
✕補助金代行業 に 流れる — 現金回収サイクル が 長すぎ、 元請け が 他社 に 奪われる
✕IT 導入支援事業者 登録待ち で 動き を 止める — 登録前 から 「6 月公募予定」 で 商談に 入る
✕demo 8 個 全部 出す — 投票結果 で 2-3 個 に 絞る (5/22 までに demo-gallery 暫定集計 を 見る)
✕提案書 を 作り込み すぎる — 1 枚 A3 で 十分、 詳細 は 商談 で
✕契約書 の 自前作成 — 業界テンプレ (中小機構公開) で いい
✕主顧客像 から 外れた 工務店 に 時間 使う — 10 人超え・年商 5 億超 は 時間泥棒、 二代目 の DX 提案 で 別世界
✕「父親社長」 単独説得 を 狙う — 必ず アトツギ を 巻き込む
