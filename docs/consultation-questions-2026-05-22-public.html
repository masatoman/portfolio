Q9 (補助金スキームの 3 業種パートナー紹介)
補助金訴求 (実質負担を抑える提案) を商談武器化 + 申請実務まで一気通貫でご提供するには 「税理士 + 行政書士 + 中小企業診断士」 の 3 業種パートナー が必要だと考えています。 多摩地区で工務店向けに共同提案できる方を、 以下 3 業種でご紹介可能ですか?

  • 行政書士 — 補助金申請書類作成・採択後成果報告の実務担当
