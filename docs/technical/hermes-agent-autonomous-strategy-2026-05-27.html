何ができるようになるか

1. 寝てる間に動く AI 秘書

夜 23 時に Telegram で「明日の面談資料まとめて」→ hermes が docs/ を検索 → Claude API で要約 → 結果を Telegram に送信。朝スマホを見たら完了している。

2. 工務店 BPaaS の原型

アトツギが Telegram で現場写真を送信 → 自動リネーム + フォルダ整理 + 日報ドラフト生成。月 3-8 万で売れるサービスがそのまま動く。

3. 定期レポート自動生成

毎週月曜 9 時に git log + GSC データ + ヒアリング進捗を自動集計 → Telegram に週次レポート送信。Mac 起動不要。

4. マルチクライアント並列

3 社のアトツギが同時に Telegram で依頼 → セッション分離で並列処理。Claude Code は 1 人しか使えないが、hermes は複数顧客に同時提供できる。
