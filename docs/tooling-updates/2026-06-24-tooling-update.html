06-22 フィードに未掲載だった同日リリース。 claude mcp login <name> / claude mcp logout <name> コマンドで MCP 認証をターミナルから直接管理できるようになった。 セッション中に ! プレフィックスで bash コマンドを入力すると Claude が自動応答するようになった（無効化は "respondToBashCommands": false）。 /plugin の Installed タブに「Skills」セクションが追加され、インストール済みスキルを一覧確認できるようになった。 /workflows にステータスフィルターが追加され、複数 routine 並走時の監視が楽になった。
