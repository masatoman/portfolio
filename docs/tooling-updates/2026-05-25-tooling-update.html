軸 B 適用 Claude Code /usage コマンド — スキル・サブエージェント別使用量内訳 v2.1.149 / 2026-05-22

/usage コマンドでスキル・サブエージェント・MCPサーバー・プラグイン別の使用量内訳が確認できるようになった。 daily-operator / night-writer / issue-finder daily queue など軸 B の自動ルーチン群が MAX プランの何割を消費しているかをコマンド一発で可視化できる。 どのルーチンがコストを食っているか数値で把握し、軽量化 or 廃止判断の根拠として使える。

推奨アクション: 次回 daily-operator 実行後に /usage を叩き、サブエージェント別消費量を確認する。消費が多いルーチン (特に Explore agent を多用するもの) を特定 → 使用頻度見直しに使う。
