🟢 即使う Claude Code Releases Claude Code v2.1.196 — /code-review トークン ~25% 削減 + streaming watchdog デフォルト ON

2026-06-29 23:27 リリース。主要変更点は 3 つ。 ① /code-review が ~25% トークン削減 — cleanup finder 5 本を統合した結果、 コードレビュースキル実行コストがそのまま下がる（MAX 定額でも token 消費≒レイテンシ削減として体感できる）。 ② streaming idle watchdog がデフォルト ON — 5 分間イベントなしで自動 abort/retry するため、 この日次フィードルーティン自体を含む長時間バックグラウンドジョブのハング問題が自然に解消される。 ③ セキュリティ修正 — claude mcp list/get が未コミットの .claude/settings.json を読んで MCP サーバーを起動しない。信頼されていない workspace は ⏸ Pending approval 表示に変わる。 その他: 組織デフォルトモデルをコンソールで設定可能に、セッション開始時に可読な名前を自動付与、 ファイル添付を Cmd/Ctrl-Click でローカルのファインダーから開けるように。

効率厨 verdict: /code-review スキルを日常的に呼ぶ masatoman のワークフローでトークン消費が黙って 25% 減る。 watchdog ON で「ルーティンがハングして朝気づかなかった」ケースが自動リトライに変わる。 何も設定変更せず claude update するだけで両方効く。
アクション: claude update を実行するだけ。watchdog は設定不要でデフォルト ON。
