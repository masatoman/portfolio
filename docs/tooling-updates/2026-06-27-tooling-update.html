🟢 即使う Claude Code v2.1.195 Claude Code v2.1.195 (2026-06-26) — ハイフン付き hook マッチャーのバグ修正 / 日本語音声入力の自動送信 fix / Remote セッション起動改善

ハイフンを含む hook 識別子（例: code-reviewer、mcp__brave-search）が substring-match になっていたバグを修正。 code-reviewer として登録した hook が他の名前でも誤ってマッチしていた可能性がある。今後は完全一致（exact-match）になる。

日本語・中国語・タイ語など単語間にスペースがない言語で、音声入力の自動送信（auto-submit）が機能しないバグを修正。 macOS の音声入力で日本語を使っているなら今すぐ恩恵を受けられる。

プロジェクトの .claude/settings.json で有効にした外部プラグインが、毎回ローダーパスを通るたびにインストール同意を求めてくる問題を修正。 ルーティン起動のたびに同意ダイアログが出ていたなら今後は出なくなる。

その他: CLAUDE_CODE_DISABLE_MOUSE_CLICKS 環境変数追加（フルスクリーンモードでマウスクリックを無効化）/ claude agents のバックグラウンドジョブが新しいバージョンで書き込まれた際に消える問題を修正 / Remote セッション起動時にプロビジョニングチェックリストを表示するよう改善 / Linux での「マイクなし」と「SoX 未インストール」の区別改善。

効率厨の一言: hook matcher の完全一致修正は code-reviewer hook や mcp__ prefix 付き hook を使っている routine に直接影響する。 設定変更は不要（修正は自動適用）—claude update で v2.1.195 に上げるだけ。 日本語音声入力の fix は、音声 + 日本語で使っている場合に今日から体感できる。

🚫 罠 GitHub Trending Agent-Reach (v1.5.0 / 42k stars) — AI エージェントに Twitter / Reddit / YouTube 等のネット閲覧を付与するツール

AI エージェント（Claude Code / Cursor 等）が Twitter、Reddit、YouTube、GitHub などを読み書きできるようにする CLI ツール。 単一コマンドでインストールでき、Web スクレイピング・YouTube トランスクリプト取得・RSS 購読・Exa セマンティック検索などを統合している。MIT ライセンス。
