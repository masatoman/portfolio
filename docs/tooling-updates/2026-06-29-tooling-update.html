🚫 罠 Google / June 18 〜 停止 Gemini CLI 個人向け無料枠 終了 — 「無料でコーディング補助」は消えた

2026年6月18日以降、Gemini Code Assist for Individuals・Google AI Pro・Google AI Ultra 向けの Gemini CLI がリクエストを受け付けなくなった。 影響ユーザーは「Antigravity CLI」への移行が必要（Google 公式案内）。 残った無料枠は Gemini Flash ベースで 1,000 req/day だが、 1 つのプロンプトが内部で数十の API コールを消費するため 「1,000回プロンプトできる」は誤解——実際はすぐ上限に当たる。

効率厨 verdict: masatoman は Claude Code MAX 定額なので直接影響なし。 ただし「Claude Code は高いから一部タスクを Gemini CLI で節約していた」 という開発者仲間・読者がいれば共有価値あり。 「Gemini 無料でコーディングできる」という情報を今でも信じている人は移行先を要確認。 代替として Gemini CLI 自体は無料枠が残るが、Flash ベースで実用的なコーディング補助には力不足。
