軸 B 工務店プロジェクトに使える更新

軸 B 適用 Stop / SubagentStop Hook が additionalContext を返却可能に v2.1.163 / 2026-06-04

Stop / SubagentStop フックが hookSpecificOutput.additionalContext を返せるようになった。 Claude はこれを次ターンの追加コンテキストとして受け取り、フックエラーとは扱われない。 これにより「ルーティン終了後に次回実行時刻・残タスク数・判定結果」などを Claude に渡せる。

推奨アクション: 既存の Stop hook スクリプト (issue-finder daily queue / pdca-weekly / daily-operator) に additionalContext フィールドを追加し、次ルーティン起動時のブリーフィング文を返すよう改修する。 具体的には echo '{"hookSpecificOutput":{"additionalContext":"..."}}' 形式で返す。 工数目安: 既存 hook 1 本あたり 15 分。
軸 B 適用 クロスセッション・メッセージングのセキュリティ強化 v2.1.166 / 2026-06-06

SendMessage 経由で他セッションからリレーされたメッセージは、ユーザー権限を失い、 受信側によるリレー許可リクエストも拒否されるようになった。 オートモードでも同様にブロックされる。

推奨アクション: night-writer / tuesday-watchdog / daily-operator 等の並列ルーティンで、 別セッションから SendMessage 経由でツール権限を付与しているケースがないか確認する。 既存フック設定に変更は不要だが、「ルーティン間連携」を新規追加する場合は権限降格を前提に設計する。
軸 B 適用 バックグラウンドセッション: 更新後の再接続時タスク消失を修正 v2.1.163 / 2026-06-04

バックグラウンドセッションが Claude Code 更新後に再接続した際、実行中タスクが失われるバグを修正。 更新後にバックグラウンドエージェントが自動で新バージョンに切り替わるようにもなった。 night-writer / zero-to-one-diary 等の長時間ルーティンの中断リスクが低下する。

推奨アクション: 特段の設定変更は不要。 今後バックグラウンドルーティンが途中で止まっていた場合の原因として「バージョン更新時リセット」を除外できるようになった。 ログ確認時の切り分けに活用する。
軸 B 適用 Hook の if 条件 ($HOME / $VAR を含むコマンド) バグ修正 v2.1.163 / 2026-06-04

Hook の if: "Bash(...)" 条件式に $HOME や $VAR を含む場合、 常にフックが発火してしまうバグを修正。 また $HOME を経由したホームディレクトリパスへの deny ルール適用も修正された。

推奨アクション: 既存フック設定 (.claude/settings.json) の if 条件に $HOME / ${VAR} を使っている箇所があれば、 意図した条件で発火しているか動作確認を行う。 修正前に意図せず全発火していた可能性がある。

Obsidian Vault 育成に使える更新

Obsidian 適用 MCP ツール deny ルールにグロブパターン対応 v2.1.166 / 2026-06-06

deny ルールのツール名にグロブパターンが使えるようになった ("*" で全ツール拒否、 "mcp__Supabase__*" のような prefix ワイルドカードも使用可能)。 allow ルールは非 MCP グロブを拒否し、起動時に未知ツール名を警告表示する。

推奨アクション: Obsidian Vault の memory / reports フォルダ保護で、 mcp__Notion__* や mcp__Google_Drive__* 等を一括 deny する設定を .claude/settings.json に追加することで、 外部サービスへの意図しないファイル書き出しを防止できる。 既存の PreToolUse/Edit-Write フックと役割分担を整理する契機にもなる。

※ Obsidian 公式リリース (obsidian-releases GitHub): 過去 7 日間のリリースなし。最新は v1.12.7 (2026-03-23)。

MAX プラン定額内運用では思考トークンのコスト最適化が不要。ルーティンのレイテンシ削減が必要になったら再検討。
