# 事業計画書 v5 — 工務店アナログ業務運用支援

> **Hero Statement**: 「ANDPAD で詰まった工務店向けに、 FAX / 電話 / LINE どれでも OK の運用支援を、 月 ¥3-5 万円から」

## 1. 事業概要

- **事業名**: 工務店アナログ業務運用支援 (BPaaS + 運用支援 + ツール提供)
- **事業形態**: 個人事業主 (Year 2 で合同会社設立検討)
- **事業内容**: 中小工務店向けに、 FAX / 郵送 / 電話 を主軸にした業務運用支援 + 必要時代行サービスを提供。 既存 SaaS (ANDPAD 等) で詰まった工務店オーナー (60-70 代 IT 弱者) を主顧客とし、 「DX しなくていい仕組み」 を月 ¥3-5 万円から提供する
- **競合との差別化 5 軸**: ① FAX バイラル拡散 / ② 月 ¥3-5 万円帯 / ③ ANDPAD 詰まり層 / ④ 「DX しなくていい」 反主流ポジ / ⑤ 地理依存ナシ

## 2. 主顧客像 (3 層構造)

| 層 | 具体像 | 特徴 + 通信手段 |
|---|---|---|
| 決裁者 | 社長 60-70 代 | IT 弱者。 LINE もメールも苦手。 **FAX / 電話 / 訪問 / 郵送** が業務の中心 |
| 経理担当 | 奥さま 60 代 | 領収書整理 / 請求書発行 / 確定申告で月末月初パンク。 紙の帳簿中心 |
| 推進役 | 2 代目 / 3 代目 30-40 代 | スマホは普通に使える。 「父親に説得材料を渡したい」 のが本音 |

## 5. 市場分析

- **TAM**: 全国中小工務店 (1-5 人規模) 約 5 万社

- **SOM**: 5 年で多摩 5% シェア = 75-100 社、 月商 ¥3-5M、 年商 ¥36-60M

## 9. 3 段階ロードマップ

### 第 1 段階 (2026-05 〜 09) — 軽量導入
- ① 写真 + ② 議事録 + ③ ANDPAD を 1-3 社で試験提供
- 友人 N=2-3 ヒアリング深掘り
- 建材屋営業マン経由で工務店紹介
- 5 月末-6 月初旬 公庫融資 100 万円 申込
- 6 月中 五反田引越し検討中
- **目標**: 1-3 社契約 / 月次売上 ¥30K-150K

### 第 2 段階 (2026-10 〜 12) — 運用支援強化
- ④ 経理 + ⑤ 補助金 をアップセル
- 認定支援機関税理士 + 行政書士 提携完了
- IT 導入補助金 (通常枠 1/2) で 1-2 件採択
- Pマーク取得
- **目標**: 月次売上 ¥100K-300K / 商品方向性確定

### 第 3 段階 (2027-01 〜 04) — 地域インフラ化
- ⑥ 地域相談窓口 を本格発動
- 信金 / 商工会議所と委託契約獲得
- 業界紙取材で信用補強
- **目標**: 3-5 社契約累計 / 月次売上 ¥300K-600K

## 10. リスクと対策 (プレモータム 5 + メタ 1)

| # | リスク | 対策 |

| ① | ヒアリング偽陽性 (受注ゼロのまま夏が過ぎる) | 友人 N=2-3 で「今日決める?」 コミット質問。 「親方に聞く」 → 48h 以内親方面談 |
| ② | オペが 10 社目で詰まる (一人運営の物理上限) | 3 社で工数 25-48h、 5 社以上は副業外注 (NDA 締結) 必須 |
| ③ | 写真機密問題で 1 社目導入が止まる | 契約書テンプレに「施主同意取得は工務店側責任」 明記 |
| ④ | 「LINE 送るだけ」 が学習コスト化 | v5 で FAX / 郵送 / 電話主軸 = 学習コストそのものを発生させない |
| ⑤ | 大手 SaaS が代行付きプラン参入 | ANDPAD 非対応 / 詰まり層特化ポジ + 地域密着 + 信用補完 |
| ⑥ | 戦略ピボット連鎖 (6 日で 4-5 ピボット) | 戦略凍結宣言 + 北原氏第三者証人化 |

## 11. 戦略凍結宣言

- **凍結期間**: 2026-05-23 〜 2026-11-22 (6 ヶ月)
- **凍結対象**: 商品仮説 + ターゲット顧客像 + 価格構造 + 差別化軸
- **凍結対象外**: 地理 + 紹介経路 (五反田引越し検討中のため柔軟)
- **解除条件**: 月次 KPI 未達 3 回連続でのみ戦略再考権発動
- **第三者証人**: 2026-05-22 北原氏面談で宣言、 経営支援のプロを証人化

## 12. 法人化方針

- **節税ルート (マイクロ法人)**: 2026-05-13 撤回済 (年 28-38 万維持費が検証フェーズに重い、 小規模企業共済で十分)

## 14. 結論

本事業は仮説段階だが、 オペレーション実現可能性は確認済み (一人運営で 3 社まで、 公庫融資 100 万円で初期投資完結)。 競合 5 社存在の中で、 5 軸差別化 (FAX バイラル / 月 ¥3-5 万 / ANDPAD 詰まり層 / 反 DX ポジ / 地理依存ナシ) で参入可能。 戦略凍結期間 6 ヶ月で集中実行し、 11/22 時点で月商 ¥30 万を達成、 24 ヶ月後の単独主軸化 (生活費 6-10 割) を目指す。
