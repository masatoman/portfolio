02ターゲットペルソナ必須

役職 / 立場

  • 主: masatoman の友人 (工務店現場監督、 30〜40 代、 男性、 LINE と Excel と電話が業務 IT の全部)
  • 副: 友人紹介で広がる 5〜30 人の中堅工務店オーナー (社長 + 息子 + 職人 5〜10 人)

流入経路

  1. masatoman が個別に LINE / 名刺 / Google Meet で URL を渡す (5/15 ヒアリング含む)
  2. 中期的に SEO + Google ビジネスプロフィール (まだ未着手)
  3. 工務店向け業界誌・展示会 (将来)

見た瞬間の心理

  • 「Web 屋 多すぎる、 みんな同じに見える」
  • 「ANDPAD は高すぎ、 freee は工務店向けじゃない」
  • 「自分の現場、 分かってくれる人いるんかな」

このページで判断したいこと

  • 「この masatoman って人は工務店のこと分かってる人か?」
  • 「自分の現場の困りごと (日報 / 写真 / 図面 / 領収書) に効くか?」
  • 「予算的に頼めるか? (5 万までなら即決、 30 万以上は社長相談)」

04訴求軸 (3〜5 個)必須

順序が重要 (上ほど見出し / 太字に近い扱い)。 saas-pricing.md と customer-target.md を踏まえた訴求。

順軸
1「工務店専門の Web + 業務改善 を 一人で全部」 — ANDPAD 入れるほどじゃない 5〜30 人規模に絞り、 masatoman 個人が直接サポート
2「価格は 初期 10 万 + 月 1 万 から」 — 業界相場 (ANDPAD 数十万 + 月 3 万、 KENTEM 系 月 2 万) より下、 個人親方でも続けられる
3「8 種類の 触れる制作例」 — 抽象論なし、 全部 dev で動くデモ。 友人ヒアリング 5/15 用
4「現場の言葉でしか書かない」 — 「DX」「SaaS」「アジャイル」 ゼロ。 「日報」「電話」「写真」「領収書」「図面」 で話す
5「公開後も ずっと面倒みる」 — 受託で売り逃げではなく、 月額で関係継続

05避けたい印象 / NG ワード必須

避けたい印象

  • Y2K / ゲーム会社風 (= /portfolio の世界、 トップに混ぜない)
  • 大企業向け SaaS の整いすぎた感じ (= ANDPAD 風、 個人親方が萎縮する)
  • 「AI で全自動」 を匂わせる (= ai-positioning.md、 工務店オーナーは流行り物に距離を置く)
  • 補助金で半額を強調 (= subsidies-positioning.md、 IT 導入支援事業者未登録なので)

NG ワード

DXアジャイルスプリント SaaSMVPSDK API認証DB Next.jsSupabase モバイルデバイスソリューション オンボーディング

触れない方が良いこと

  • 「他の Web 屋が高い」 を直接言わない
  • マイクロ法人設立 (2026/6) は社員を持たない零細工務店オーナーの感性に合わない
  • 個人開発作品 (recipe-ai 等) は /lab 送り、 トップから見せない (= works-strategy.md)

  • BusinessContactForm (Resend 連携済み) を Contact セクションで使う
  • /local-business/<demo> への内部リンク 8 本は変更しない

ブランド制約

  • 屋号: Masato Works (ヘッダーロゴ)
  • 個人名: 井原誠斗 (ihara masato)
  • メインカラー: brown #7a5c38、 アクセント #a88456、 背景 #f3efe7 / #f8f5ee、 テキスト #1f2a37
  • フォント: Noto Serif JP (見出し)、 Inter (本文)
  • 既存ロゴ「M」は丸い白背景 + brown 文字 (現状維持)

法令 / 規約 / 倫理

  • 価格は「初期 10 万 + 月 1 万 から」 と書き、 「内容により応相談」 を併記
  • 「全自動」「AI で完璧」 のような誇大訴求 NG

データ制約

  • savings-simulation の数字は「概算」表記。 5/15 友人ヒアリング後に実測値で差し替え予定 (今回 LP リデザインの範疇外)

A付録 A: 推奨セクション構成 (現状 vs 提案)

順現状提案変更理由
1 Hero (画像 + テキスト + CTA 2 個) Hero (写真 + 価格バッジ + 1 行訴求 + LINE/相談 CTA) 価格を Hero に出すと工務店オーナーが即判定可、 3 秒で「自分のため」を加速
