📌 プラン B 目的

友人に 1 本だけ低負荷 LINE を送って反応を見る。 反応があれば 後付けで N=1 化 + 紹介芋蔓 1 件目起点として復活。 反応なしで自動的にプラン C に移行。

🎯 LINE 送付の前提条件

  • 1 本のみ、 リマインドなし
  • 「返事しなくていい」 を明示
  • 後継層関与の有無を 軽く確認するだけ
  • 友人に「忙しいのに頼まれた」 と思わせない文体

LINE 送付内容 + タイミング

送付タイミングは 5/24 (土) 朝-昼。 平日夜は仕事の続きを連想させるので避ける、 土日朝-昼は仕事と分離しやすい。 文体は「忙しいのに無理させない」 を最重要視、 5/14 ヒアリングの感謝 + 軽い確認 + 「返事しなくていい」 のセット。

送付内容案 (主顧客モード適用 — 友人は工務店現場監督)

この間は ヒアリング 話してくれて ありがとう。 あれから 友達の話を考えて、 もう少し聞きたいことが 出てきたんだけど、 仕事が落ち着いてから で全然 OK。 ひとつだけ 軽く確認、 友達の会社って 後継ぎ (2 代目 / 3 代目) っているの? それとも 親方は ずっと現役? 忙しいタイミングだと 思うので、 返事なくても 大丈夫。 落ち着いたら ご飯でも 行こう。

📌 LINE 内容の設計意図

  • 「ヒアリング」 という単語: 5/14 共有済の用語なので OK (一般的な「顧客ヒアリング」 ではなく「話を聞かせてもらった」 の意)
  • 「もう少し聞きたいこと」: 詳細を書かない (後継層関与確認 + 訪問アポ依頼を一気に書くと圧)
  • 「ひとつだけ 軽く確認」: 1 質問のみ、 答えが Yes/No で済む
  • 後継ぎ確認: 5/24-31 プラン分岐の核 (後継層関与なし = N=1 カウントせず、 別ルート)
  • 「返事なくても 大丈夫」: 圧を消す、 友人に「無視していい」 安心感
  • 「ご飯でも 行こう」: 関係性維持の意図、 仕事の話を続ける意図がないシグナル

反応シナリオ別 動き

① 即返事 (5/24-26) — 確率 10-20%

反応例: 「うちは 2 代目で俺がやってる」「親方はまだ現役」 等

  • 後継ぎ Yes: 友人本人がアトツギ → プラン A 復活検討、 6 月前半訪問アポ調整、 N=1 化
  • 後継ぎ No (親方現役 + 友人雇われ): 友人 N=1 カウントしない、 ただし関係性継続、 「親方や同業の後継ぎを知ってたら 1 人だけ繋いでくれない?」 を 2 週間後に追加打診
  • 後継ぎ不明: 詳細聞かず受領 (友人圧をかけない)、 7-8 月で機会あれば再確認

② 遅返事 (5/27-31) — 確率 10-15%

反応例: 「ごめん 返事遅れた、 ◯◯◯」

  • 後継層関与確認のみ受領、 訪問アポは 提案しない (友人忙しい)
  • 「忙しい時にありがとう、 落ち着いたら ご飯行こう」 のクッション返信のみ
  • プラン C は並走継続、 友人扱いは「業界知識保持」 のまま
  • 7-8 月で機会あれば再アプローチ (友人発信であれば乗る、 こちらから送らない)

③ 無反応 (5/31 まで) — 確率 50-70%

反応: 既読つかず or 既読のみ

  • 5/31 23:59 でプラン C 確定発動
  • 友人 LINE は閉じる (リマインド送らない、 6 月以降も自発反応待ち)
  • 5/14 N=1 結果は業界知識として温存、 ヒアリング N=1 カウントせず
  • プラン C 6 経路並列を本格起動 (北原氏 + たましん は既に 5/24-31 で接触開始済)

④ 後日反応 (6 月以降) — 確率 5-10%

反応例: 「久しぶり、 落ち着いた」 等の自発連絡

  • この時点で他経路で N=2-3 達成済の可能性
  • 友人発信なら乗る、 自然な会話の流れで後継層関与確認 + 訪問依頼
  • 7-8 月で N=1 化 + 紹介依頼 (5/22 北原氏「紹介芋蔓式」 を友人雪だるまに再接続)
  • こちらから 6 月以降 LINE 送らない: 友人の生活ペースを尊重、 機会あれば乗る程度

📌 確率の想定根拠

  • 長期忙しい (半年以上) = 即返事 + 遅返事 合計 20-35%、 主流は無反応 50-70%
  • 5/14 ヒアリング当日は普通に答えてくれた = 関係性は維持、 後日反応 5-10% は十分あり得る
  • 圧をかけない設計: 返事ハードルを下げることで 即返事率が 5-10% 上がる期待値
  • ※ 確率は仮説、 5/31 実績で更新
