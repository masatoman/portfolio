01前回 メモ から 既に 分かった こと

カテゴリ別 に 整理。 これは これ以上 聞かなくて OK な 確定事項 として、 当日 は 「念のため 確認」 程度 に。

現場監督 の 1 日

  • 現場 と 事務所 が 片道 2 時間 以上 (現場事務所 なし)
  • 業務: 図面 / 工程表 / 見積書 作成、 現場で 使える ツール が欲しい
  • 大工 / 電気 / 水道 / ガス の 各業者 を 取りまとめ = 「人と人 を 取り持つ 仕事」
  • 大工 と は 上下関係 発生 (ほぼ言いなり)、 監督 が 我慢する 構図
  • 区役所・警察署 への 申請 = 根回し が 必須 (コロナ 以降 一部 郵送 化)

受発注 の 構造

  • 料金表 なし、 坪単価 で 話す (200 万 で 「安い」 の 感覚)
  • 3 社 見積もり 慣習 → 「他社 より 高い と やだ」 の プレッシャー
  • 客 が 「1 億 で 」 と 上限 指定 → 設計が グレード を 下げて 調整
  • 「便器 を 交換」 みたい な 単純発注 = ものを 買う 値段 で 終わる ケース
  • 3 ルート: 客 → 設計事務所 / 客 → 工務店 / 客 → ハウスメーカー

業界 の 構造

  • プロジェクト 種類: 住宅 (メイン) / マンション / アパート / 神社 / 病院 / リフォーム
  • 設計事務所 = ハブ。 コンペ で 選ばれる、 長い 付き合い と 毎回違う が 混在
  • 大工 にも 専門分野 (床だけ / 内装 / 全部できる) が 存在
  • 木造 = 大工、 鉄筋コンクリート = 内装 にだけ 大工 が 入る

IT 受容性 / 業務ツール

  • みつもろう 17 を 使用、 PDF → 手動で 入力し直し (二重作業)
  • LINE 通話 を 業務で 普通に 使う
  • 090 → 文字起こし テスト を 試した こと あり (関心 度 高い)

主顧客 は 「ものを 買う 値段」 で 動く (補助金 = 「金 もらえる」 動機) ◎ 一致 「便器 を 交換します = ものを 買う 値段」 補助金 案件 で 一人親方 が 集まる 場 は 実際 ある?

商工会議所 異業種交流会 には 来ない (中堅以上 のみ) △ 未検証 あなた (現場監督) の 周りで 商工会議所 行く 人 いる? 印象 は?
建設組合 (東京土建 / 全建総連) は アクティブ △ 未検証 友人 メモ に なし あなた や 周り の 親方 で 東京土建 入って る 人 いる? 集会 行く?
業界紙 (新建ハウジング) は 紙 で 読む △ 未検証 あなた や 親方 は 新建ハウジング / 日刊建設工業 読む?
飛び込み 営業 は 信用 されない、 紹介 が 圧倒的 (51.5%) △ 未検証 あなた (監督) は 業者 を どう 探して る? 紹介 経由 が 多い?
